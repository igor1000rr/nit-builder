## Что это

NIT Builder — open-source AI-конструктор сайтов, работающий на **локальных LLM** через LM Studio. Описываешь сайт простыми словами — получаешь готовый HTML-файл через 30-60 секунд.

**Отличие от Tilda/Wix/v0/Bolt:** всё работает локально. Никакого облака. Никаких подписок. Данные никуда не уходят. Работает без интернета после установки.

### Для кого

- Владельцы малого бизнеса, которым не хочется платить 1500₽/месяц за Tilda бесконечно
- Фрилансеры, делающие быстрые сайты клиентам
- Студенты с RTX 3060 и без бюджета
- Все, кто хочет сайт и владеть своими данными

### Что можно создать

16 встроенных шаблонов под самые частые запросы:

☕ Кофейни · 💈 Барбершопы · 📸 Фотографы · 💻 Портфолио разработчиков · 💒 Свадебные приглашения · 💪 Фитнес-тренеры · 🍽️ Рестораны · 📚 Репетиторы · 💅 Мастера красоты · 🔧 Автосервисы · 🎨 Хендмейд · 🎧 DJ/Музыканты · 🚀 SaaS-лендинги · 🦷 Медицинские клиники · 🧘 Йога-студии · 📄 Универсальный fallback

## Как это работает

```
Твой запрос («сайт для кофейни в Минске»)
        ↓
[Планировщик LLM] → JSON-план: {тип_бизнеса, тон, секции, цвета, template_id}
        ↓
[Подбирается подходящий шаблон из каталога]
        ↓
[Кодер LLM] → адаптирует шаблон под твой план → стримит HTML
        ↓
Живое превью в iframe · Скачать одним HTML-файлом
```

**Почему адаптация шаблона, а не генерация с нуля:**
Маленькие локальные модели (7B параметров) плохо справляются с написанием полного React-проекта с импортами, компонентами, конфигами. Но отлично справляются с *адаптацией готового HTML*. Именно это делает NIT Builder работоспособным на RTX 3060.

## Быстрый старт

### Требования

- **Node.js 20+**
- **Один из LLM-провайдеров:**
  - **LM Studio** (рекомендуется, бесплатно, локально) — [lmstudio.ai](https://lmstudio.ai)
  - **Groq** (бесплатный tier, облако) — [console.groq.com](https://console.groq.com/keys)
  - **OpenRouter** (платный, облако) — [openrouter.ai](https://openrouter.ai/keys)

### Установка

```bash
git clone https://github.com/igor1000rr/nit-builder.git
cd nit-builder
npm install
cp .env.example .env
# Отредактируй .env — задай LMSTUDIO_BASE_URL или GROQ_API_KEY
npm run dev
```

Открой [http://localhost:5173](http://localhost:5173).

### Локальный режим (LM Studio)

1. Скачай [LM Studio](https://lmstudio.ai)
2. Скачай модель: **Qwen2.5-Coder-7B-Instruct** (Q4_K_M, ~4.5 ГБ)
3. Запусти локальный сервер в LM Studio (Developer → Start Server)
4. Проверь `LMSTUDIO_BASE_URL=http://localhost:1234` в `.env`
5. `npm run dev` → опиши сайт → готово через ~60 секунд

### Облачный режим (без GPU)

1. Получи бесплатный Groq API ключ
2. Задай `GROQ_API_KEY=gsk_...` в `.env`
3. `npm run dev`

## Стек

- **React Router v7** (SSR) + **React 19** + **TypeScript**
- **Tailwind CSS v4** через Vite plugin
- **Vercel AI SDK** (`ai` + `@ai-sdk/openai`) для стриминга LLM
- **Zod** для валидации плана
- Генерируемые сайты используют **Tailwind CDN** + **Alpine.js CDN** — без сборки

Всего ~5000 LOC. Маленький, читаемый, хакабельный кодбейз.

## Контрибьюшн

### Добавить новый шаблон

Шаблоны — это обычные HTML-файлы в `app/templates/html/`. Чтобы добавить свой:

1. Создай `app/templates/html/your-template-id.html`
2. Правила:
   - Один файл, от `<!DOCTYPE html>` до `</html>`
   - Tailwind через CDN
   - Alpine через CDN для интерактивности (опционально)
   - Картинки — только Unsplash или inline SVG
   - Без локальных ассетов, без npm-пакетов
   - Адаптивность (классы `sm:`, `md:`, `lg:`)
3. Добавь метаданные в `app/lib/config/htmlTemplatesCatalog.ts`
4. Открой PR со скриншотом

Смотри существующие шаблоны (`coffee-shop.html`, `portfolio-dev.html`) как образец.
